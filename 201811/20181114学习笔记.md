| 值         | 描述                       |
| ---------- | -------------------------- |
| none       | 用户无法调整元素的尺寸     |
| both       | 用户可调整元素的高度和宽度 |
| horizontal | 用户可调整元素的宽度       |
| vertical   | 用户可调整元素的高度       |
